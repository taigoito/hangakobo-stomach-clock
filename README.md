# HANGAKOBO: STOMACH CLOCK

おさのなおこの版画で作られた時計。その名も「腹時計」。
ユーモラスなテーマに沿って構成された、インタラクティブなシングルページ作品です。  
フルスクラッチで実装されたアナログ時計のアニメーションや要素配置を通して、動きの遊びやテンポ感を試験的に探りました。

---

## 特徴

- 現在時刻を示すアナログ時計
- ミニマルで軽量な構成
- HTML / CSS / JS のみで制作
- PC表示に最適化
